# ⚡ VIT Smart Energy Utility Dashboard

A Streamlit-based dashboard for monitoring and analyzing
department-wise and floor-wise energy consumption in a college campus.

## 📌 Features
- Department-wise energy analysis
- Floor-wise energy consumption
- Component-level breakdown (Classroom, Washroom, Water)
- Sustainability indicators (CO₂ emission & solar offset)
- Clean, responsive UI using Streamlit containers

## 🛠 Tech Stack
- Python 3.13
- Streamlit
- Pandas
- Matplotlib

## ▶️ How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
